# Brewery Notes

These are working records for planning brews. They deliberately live outside `recipes/`, which is the Zensical `docs_dir`, so they are committed to the repository but not published on the recipe site.

- [`ingredients.md`](ingredients.md) records ingredients currently available.
- [`brewing-method.md`](brewing-method.md) summarises the usual equipment and process that recipes should be designed around.

Update the inventory after each brew or ingredient purchase. Quantities marked approximate should be weighed before finalising a recipe.
